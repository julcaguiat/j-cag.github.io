Enterprise Network, Systems & Security Operations Environment

This project is a continuously evolving enterprise infrastructure and security operations environment designed to simulate real-world IT and cybersecurity operations. The environment integrates Cisco networking (VLANs, Router-on-a-Stick), Windows Server Active Directory services (AD DS, DNS, DHCP, SMB), Proxmox virtualization, isolated pentesting environment and centralized SIEM monitoring with Wazuh. Security operations are extended through Sysmon telemetry, automated incident response, and workflow orchestration using n8n and Peppermint. This project serves as a hands-on portfolio demonstrating practical experience across network engineering, systems administration, virtualization, security monitoring, and workflow automation.

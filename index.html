Enterprise Infrastructure & Security Operations Environment

This project is a continuously evolving enterprise infrastructure environment designed, deployed, and maintained to simulate the technologies, architectures, and operational workflows commonly found in modern enterprise networks and Security Operations Centers (SOCs). The environment integrates networking, systems administration, virtualization, security monitoring, and automation into a unified operational platform.

Core components include Cisco enterprise networking with VLAN segmentation and inter-VLAN routing, Windows Server Active Directory Domain Services (AD DS), DNS, DHCP, SMB file services, and Proxmox virtualization. Security capabilities include centralized monitoring and detection through Wazuh SIEM, Sysmon endpoint telemetry, automated incident response workflows, and IT service management integration through n8n and Peppermint.

The environment is used to design, validate, document, and operate enterprise technologies across multiple disciplines, including network engineering, systems administration, infrastructure deployment, security operations, detection engineering, and workflow automation. Every component is deployed, integrated, and maintained as part of an end-to-end enterprise platform that demonstrates practical experience with technologies and operational processes used in production environments.

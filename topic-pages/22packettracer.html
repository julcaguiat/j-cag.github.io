  •     Core Network Foundation

    • Lab 1: Subnetting, VLAN & Trunk Configuration, Router-on-a-Stick Inter-VLAN Routing, NTP, and Secure Device Access This lab focuses on designing and implementing a segmented network using proper subnetting, VLANs, trunking, and Router-on-a-Stick for inter-VLAN routing. This lab also introduces core management and security practices such as configuring a management VLAN, enabling NTP, and securing device access with local authentication and SSH.

    • Lab 2: Building Redundant Layer 2 Networks with STP, EtherChannel, and VTP This lab focuses on designing a redundant Layer 2 switching topology using VLANs, trunking, EtherChannel, and Rapid PVST+. It demonstrates how Spanning Tree Protocol (STP) prevents switching loops while EtherChannel provides increased bandwidth and link redundancy between switches. The lab also implements VTP for centralized VLAN management and applies best practices such as native VLAN reassignment, PortFast, and BPDU Guard to improve network stability and security.

    • Lab 3: Implementing Enterprise VLANs: VLSM, VTP, Voice VLANs, EtherChannel, and Static IP Routing This lab implements an enterprise-style campus network using best practices, including VLSM IP addressing, VTP-managed VLANs, Layer 2/3 and static EtherChannel for redundancy, voice VLANs for IP phones, a dedicated management VLAN, and native VLAN reassignment. A multi-tier core, distribution, and access topology reinforces Layer 2 design, VLAN segmentation, trunking, VoIP integration, and static routing for reliable inter-VLAN and inter-office connectivity.

    • Lab 4: Enterprise Campus Network with Redundancy, Dynamic Routing, WAN Connectivity, ACLs, and NAT/PAT This lab builds on Lab 3 by extending the enterprise campus network to include high availability, inter-VLAN security, and WAN connectivity. Key additions include ACLs to enforce access policies, NAT with PAT for controlled internet access, link aggregation via LACP, PAgP, and static EtherChannels, HSRP for first-hop redundancy, PPP with CHAP on WAN links, and OSPF across LAN and WAN for dynamic routing. The lab strengthens practical skills in scalable, secure, and highly available network design, closely reflecting modern production environments.

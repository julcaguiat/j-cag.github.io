Resetting Windows Password & Disabling PIN Login (Windows 10)

Offline password reset using Ubuntu Live USB and chntpw for local Windows accounts (lab-only procedure).

Overview

This guide demonstrates how to reset a Windows local account password and disable Windows Hello PIN authentication using an Ubuntu Live USB environment and the chntpw utility.

This technique works by modifying the Windows SAM (Security Account Manager) database offline.

Requirements

  • Second working computer
  • USB drive (8GB or larger)
  • Ubuntu Desktop ISO
  • Rufus (USB creation tool)
  • Internet access (for chntpw installation)

1. Create Ubuntu Live USB

Download Ubuntu

Download Ubuntu Desktop ISO from: Ubuntu Official Website

Create Bootable USB (Rufus)

  • Insert USB drive
  • Open Rufus
  • Select Ubuntu ISO
  • Partition scheme: MBR
  • File system: FAT32
  • Click Start

2. Boot Locked System from USB

  • Insert USB into target system
  • Power on and press ESC repeatedly
  • Press F12 for boot menu
  • Select USB (UEFI preferred)
  • Choose “Try Ubuntu” (do not install)

3. Connect to Internet

  • Connect to Wi-Fi from top-right icon
  • Verify connectivity
ping -c 2 google.com

4. Install chntpw Tool

sudo apt update
sudo apt install chntpw

5. Locate Windows Partition

lsblk -f
sudo lsblk -o NAME,FSTYPE,SIZE,LABEL,MOUNTPOINT

Example output indicates NTFS partition (Windows system drive).

Mount Windows Partition

sudo mkdir /mnt/win
sudo mount -t ntfs-3g /dev/nvme0n1p3 /mnt/win

Verify SAM File

ls /mnt/win/Windows/System32/config/

6. Reset Password with chntpw

sudo chntpw -i /mnt/win/Windows/System32/config/SAM

Inside menu:

  • Option 1: Edit user data
  • Select username or RID
  • Option 1: Clear password
  • Option 4: Unlock user (if needed)
  • Option !: Exit editor
  • Option q: Quit
  • Confirm with y

7. Disable Windows Hello (PIN Login)

Remove Windows Hello credential store:

sudo rm -rf /mnt/win/Windows/ServiceProfiles/LocalService/AppData/Local/Microsoft/Ngc

This removes PIN, fingerprint, and facial authentication data.

8. Reboot and Login

sudo reboot
  • Remove USB when prompted
  • Select “Sign-in options”
  • Choose password login
  • Press Enter (blank password if cleared)

chntpw Overview

chntpw stands for “Change NT Password” and is a Linux utility used to modify Windows local account data by editing the SAM registry hive offline.

Key Functions

  • Reset or clear local Windows passwords
  • Unlock user accounts
  • Promote users to administrator
  • Edit SAM registry entries offline

Important Notes

  • Does not decrypt passwords (it clears them)
  • Does not work on Microsoft cloud accounts
  • Requires offline access to Windows disk

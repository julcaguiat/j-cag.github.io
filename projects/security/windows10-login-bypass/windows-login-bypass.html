OFFLINE WINDOWS RECOVERY LAB

Windows Password Reset (Rufus + chntpw)

Offline password reset and Windows Hello PIN removal using Ubuntu Live USB and chntpw in a controlled SOC lab environment.
NOTE: This is for educational purpose only!

Overview

This procedure demonstrates offline modification of the Windows SAM database using Ubuntu Live USB. It is used in lab environments to simulate endpoint recovery and credential reset scenarios.

Requirements

  • Second working system or VM host
  • USB drive (8GB+)
  • Ubuntu Desktop ISO
  • Rufus USB creation tool
  • Internet access for package installation

Step 1 — Create Ubuntu Live USB

Download Ubuntu

Download ISO: Ubuntu Desktop Download

Rufus Configuration

  • Select Ubuntu ISO
  • Partition scheme: MBR
  • File system: FAT32
  • Start USB creation

Step 2 — Boot Target System

  • Insert USB into target system
  • Press ESC/F12 for boot menu
  • Select USB (UEFI)
  • Choose “Try Ubuntu”

Step 3 — Install Required Tools

sudo apt update
 sudo apt install chntpw

Step 4 — Locate Windows Partition

lsblk -f
 sudo lsblk -o NAME,FSTYPE,SIZE,LABEL,MOUNTPOINT

Mount Windows Disk

sudo mkdir /mnt/win
 sudo mount -t ntfs-3g /dev/nvme0n1p3 /mnt/win

Verify SAM File

ls /mnt/win/Windows/System32/config/

Step 5 — Reset Windows Password

sudo chntpw -i /mnt/win/Windows/System32/config/SAM

Inside menu:

  • Edit user data
  • Select account
  • Clear password
  • Unlock user (if needed)
  • Write changes and exit

Step 6 — Disable Windows Hello PIN

sudo rm -rf /mnt/win/Windows/ServiceProfiles/LocalService/AppData/Local/Microsoft/Ngc

This removes stored Windows Hello credential data including PIN authentication.

Step 7 — Reboot System

sudo reboot
  • Remove USB when prompted
  • Select password login option
  • Log in with cleared or updated credentials

chntpw Overview

chntpw (Change NT Password) is a Linux utility used to modify Windows local account SAM database offline.

Capabilities

  • Reset local Windows passwords
  • Unlock accounts
  • Modify SAM registry hive

Limitations

  • Does not work on Microsoft accounts
  • Clears passwords rather than decrypting them
  • Requires offline disk access

Completion

Windows password reset and Windows Hello PIN removal has been completed in a controlled lab environment using Ubuntu Live USB and chntpw.

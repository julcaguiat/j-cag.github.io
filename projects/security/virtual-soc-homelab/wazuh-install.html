How to Install Wazuh SIEM on Ubuntu Server

Step-by-step guide to installing Wazuh SIEM (Manager, Indexer, Dashboard, and Filebeat) on an Ubuntu Server VM for SOC home lab monitoring.

Overview

Wazuh is a Security Information and Event Management (SIEM) platform used for log analysis, threat detection, and endpoint monitoring. In this lab, Wazuh will be deployed on Ubuntu Server to centralize security telemetry from Windows and Linux endpoints.

This setup simulates a real SOC environment where security analysts monitor events, alerts, and system activity from a centralized dashboard.

Tip: Use an AI Assistant During Setup

Wazuh installation involves multiple services and configuration files. Using an AI assistant helps troubleshoot installation errors, service failures, and configuration issues quickly.

Step 1 — Install Wazuh Components

  1. Start Ubuntu-Wazuh-Server VM
  2. Log in to the system
  3. Run the following commands:
sudo apt update && sudo apt upgrade -y
sudo apt install curl apt-transport-https lsb-release gnupg -y
curl -sO https://packages.wazuh.com/4.8/wazuh-install.sh
chmod +x wazuh-install.sh
sudo ./wazuh-install.sh -a

Installation may take 10–15 minutes. After completion, save the generated admin credentials (username and password) for dashboard access.

Step 2 — Verify Wazuh Services

Check that all services are running correctly:

sudo systemctl status wazuh-manager
sudo systemctl status wazuh-indexer
sudo systemctl status filebeat
sudo systemctl status wazuh-dashboard

All services should show an active (running) state.

Step 3 — Configure Wazuh Dashboard Access

  1. Get Ubuntu server IP address using ip a
  2. Navigate to configuration directory:
sudo su
cd /
cd /etc/wazuh-dashboard
ls
nano opensearch_dashboards.yml

Update opensearch.hosts with your Ubuntu server IP address.

  1. Navigate to indexer configuration:
cd /
cd /etc/wazuh-indexer
ls
nano opensearch.yml

Set network.host to 0.0.0.0 if required.

Step 4 — Access Wazuh Dashboard

  1. Open browser on Kali Linux or Windows host
  2. Navigate to: https://<Wazuh-Server-IP>:443
  3. Accept browser security warning
  4. Log in using admin credentials

Completion

Wazuh SIEM is now fully deployed on Ubuntu Server and accessible via web dashboard. The next step is deploying Wazuh agents on Windows 10 and Kali Linux endpoints for full telemetry coverage.

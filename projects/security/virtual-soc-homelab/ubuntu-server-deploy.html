How to Install Ubuntu Server VM (Wazuh SIEM)

Step-by-step guide to installing Ubuntu Server as a VirtualBox VM for Wazuh SIEM deployment in a SOC home lab environment.

Overview

This guide covers installing Ubuntu Server 22.04/24.04 as a virtual machine on VirtualBox. This VM will host the Wazuh SIEM platform in later stages of the SOC lab. Store all VM files in D:\virtualbox-VMs for proper organization.

Tip: Keep an AI Assistant Handy

Ubuntu Server installation involves command-line configuration and network setup. Having an AI assistant available helps resolve installation and configuration issues faster.

Requirements

  • VirtualBox installed
  • Ubuntu Server ISO downloaded
  • Internet connection

Step 1 — Download Ubuntu Server ISO

  1. Visit: Ubuntu Server Download Page
  2. Download Ubuntu Server 22.04 LTS or 24.04 LTS
  3. Move ISO file to: D:\virtualbox-VMs

Step 2 — Create Ubuntu VM in VirtualBox

  1. Open VirtualBox
  2. Click New
  3. Name VM: Ubuntu-Wazuh-Server
  4. Select Linux → Ubuntu (64-bit)
  5. Attach Ubuntu ISO file

Step 3 — VM Configuration

  1. Enable Unattended Install
  2. Set Username and Password
  3. Assign Resources:
    • RAM: 2048–4096 MB (recommended 4096 MB)
    • CPU: 2 cores
    • Disk: 40 GB
  4. Click Finish

Step 4 — Post Installation Configuration

  1. Wait for installation to complete (10–15 minutes)
  2. Shut down VM after first boot
  3. Open Settings in VirtualBox
  4. Disable Floppy Drive under System → Motherboard
  5. Verify ISO is correctly mounted in Storage (IDE Controller)

Step 5 — Start Ubuntu Server VM

  1. Start the VM
  2. Login using configured credentials
  3. Verify network connectivity

Completion

Ubuntu Server is now successfully installed and ready for Wazuh SIEM deployment. The next step is installing and configuring the Wazuh security monitoring platform.

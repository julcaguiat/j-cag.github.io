Install Ubuntu Server on VirtualBox

Step-by-step deployment guide for deploying Ubuntu Server platform for Wazuh SIEM environment.

Overview

This guide covers downloading Ubuntu Server, creating a VirtualBox virtual machine, assigning system resources, and completing the initial deployment. The server will later be used as the foundation for Wazuh SIEM and security monitoring services.

What You Will Learn

  • Download Ubuntu Server installation media
  • Create an Ubuntu Server VirtualBox VM
  • Allocate CPU, memory, and storage resources
  • Configure VirtualBox settings
  • Validate server installation and connectivity

Requirements

  • Oracle VirtualBox installed
  • Ubuntu Server ISO
  • Internet connectivity
  • Minimum 40 GB available storage

Step 1 — Download Ubuntu Server ISO

  1. Navigate to the Ubuntu Server download page.
  2. Download Ubuntu Server 22.04 LTS or 24.04 LTS.
  3. Save the ISO file to the VM storage location.

Step 2 — Create Ubuntu Server VM

  1. Open VirtualBox.
  2. Select New.
  3. Name the VM Ubuntu-Wazuh-Server.
  4. Select Linux → Ubuntu (64-bit).
  5. Attach the Ubuntu ISO.

Step 3 — Configure Virtual Hardware

  1. Enable Unattended Installation.
  2. Create a local username and password.
  3. Assign system resources.
  • Memory: 4096 MB recommended
  • Processor: 2 CPU cores
  • Disk: 40 GB

Step 4 — Post-Installation Configuration

  1. Allow the installation process to complete.
  2. Perform the initial boot.
  3. Shut down the VM.
  4. Open VirtualBox Settings.
  5. Disable the Floppy Drive under System → Motherboard.
  6. Verify storage controller and ISO configuration.

Step 5 — Verify Ubuntu Server Operation

  1. Start the virtual machine.
  2. Log in using the configured credentials.
  3. Verify network connectivity.
  4. Confirm the server boots without errors.

Install Complete

Ubuntu Server is now ready. Next will be installing Wazuh SEIM.

VIRTUAL MACHINE DEPLOYMENT

Install Kali Linux on VirtualBox

Step-by-step deployment guide of a Kali Linux virtual machine on VirtualBox.

Overview

This guide explains how to deploy a Kali Linux virtual machine on VirtualBox hypervisor.

What You Will Learn

  • Create a dedicated VirtualBox VM storage folder (optional: can use default folder)
  • Download the Kali Linux VirtualBox image
  • Extract the virtual machine package
  • Configure VirtualBox default storage location
  • Import and start Kali Linux

Step 1 — Create VM Storage Folder

  1. Open File Explorer.
  2. Navigate to Drive D (or preferred storage drive).
  3. Create a folder named virtualbox-VMs.

Step 2 — Download Kali Linux Virtual Machine

  1. Navigate to the Kali Linux download page.
  2. Select Virtual Machines.
  3. Download the VirtualBox image package.

Step 3 — Install 7-Zip

  1. Download 7-Zip.
  2. Install the 64-bit version.

Step 4 — Extract Kali Linux Image

  1. Navigate to the VM storage folder.
  2. Right-click the downloaded .7z file.
  3. Select Extract Here.

Step 5 — Configure Default VM Folder

  1. Open VirtualBox.
  2. Select File → Preferences.
  3. Set Default Machine Folder to your VM storage directory.

Step 6 — Import Kali Linux VM

  1. Select Machine → Add.
  2. Browse to the extracted Kali folder.
  3. Select the .vbox file.

Step 7 — Start Kali Linux

  1. Select the Kali Linux VM.
  2. Click Start.
  3. Wait for the operating system to boot.

Default Login Credentials

  • Username: kali
  • Password: kali

Install Complete

Kali Linux is now operational within VirtualBox ready for network adapter configuration later.

How to Install Kali Linux VM on Windows 11

Step-by-step guide to installing Kali Linux as a VirtualBox VM using a prebuilt image and storing it in Drive D.

Overview

This guide explains how to install a prebuilt Kali Linux VirtualBox image on Windows 11 and organize virtual machines on Drive D for better storage management. If Drive D is unavailable, use the default C drive.

What You Will Learn

  • Create and organize a dedicated VM folder on Drive D
  • Download and extract Kali Linux .7z VirtualBox image
  • Import Kali Linux into VirtualBox
  • Configure default VM storage location

Tip: Use an AI Assistant During Setup

VM installations may fail due to extraction errors, boot issues, or misconfiguration. Using an AI assistant can help troubleshoot setup issues quickly.

Step 1 — Create VM Folder on Drive D

  1. Open File Explorer (Win + E)
  2. Go to This PC → Drive D
  3. Create folder: virtualbox-VMs

Step 2 — Download Kali Linux VM

  1. Go to: Kali Linux Official Download Page
  2. Select Virtual Machines → VirtualBox image
  3. Download: kali-linux-2025.2-virtualbox-amd64.7z

Step 3 — Install 7-Zip

  1. Download from: 7-Zip Official Website
  2. Install the 64-bit version

Step 4 — Extract Kali VM

  1. Go to D:\virtualbox-VMs
  2. Right-click .7z file → Extract Here
  3. Wait for extraction to complete

Step 5 — Set Default VM Folder

  1. Open VirtualBox
  2. Go to File → Preferences
  3. Set Default Machine Folder to D:\virtualbox-VMs

Step 6 — Import Kali Linux VM

  1. Open VirtualBox → Machine → Add
  2. Navigate to extracted Kali folder
  3. Select .vbox file

Step 7 — Start Kali Linux VM

  1. Select VM in VirtualBox
  2. Click Start
  3. Wait for boot process

Login Credentials

  • Username: kali
  • Password: kali

Completion

Kali Linux is now successfully installed as a VirtualBox VM. You can now proceed with Windows 10 VM and Wazuh SIEM integration.

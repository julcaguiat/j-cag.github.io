Install pfSense on VirtualBox (Windows 11 Host)

Step-by-step guide to installing pfSense Community Edition as a virtual firewall/router in VirtualBox for your home SOC lab environment.

Overview

This guide covers installation of pfSense CE on VirtualBox. pfSense will act as the network firewall/router for your lab environment, enabling segmentation between attacker, endpoint, and SIEM systems.

Key Terms & Acronyms

  • pfSense: Open-source firewall/router based on FreeBSD.
  • VM: Virtual Machine running on hypervisor software.
  • NIC: Network Interface Card / adapter.
  • WAN: External network interface (internet-facing).
  • LAN: Internal network interface (trusted network).
  • NAT: Network Address Translation.
  • ISO: Disk image used for OS installation.

Requirements

  • VirtualBox installed on Windows 11
  • pfSense CE ISO downloaded
  • Hardware virtualization enabled (BIOS/UEFI)
  • 2 GB RAM minimum
  • 20 GB storage minimum

Tip: Use an AI Assistant

Keep an AI assistant available during installation. Firewall setups often fail due to interface misconfiguration, ISO issues, or boot errors.

Step 1 — Download pfSense ISO

  1. Go to pfSense Download Page
  2. Create account and log in
  3. Select AMD64 ISO for Virtual Machines
  4. Download installer ISO

Step 2 — Prepare ISO File

  1. Extract ISO using 7-Zip if compressed
  2. Move ISO to D:\virtualbox-VMs\pfSense
  3. Verify file integrity

Step 3 — Create pfSense VM

  1. Open VirtualBox → Click New
  2. Name: pfSenseVM
  3. Type: BSD → FreeBSD (64-bit)
  4. Attach pfSense ISO

Step 4 — Configure Resources

  • RAM: 2048 MB
  • CPU: 1 core
  • Disk: 20 GB (VDI)

Step 5 — Install pfSense

  1. Boot VM
  2. Press Enter through installer prompts
  3. Accept defaults for disk partitioning
  4. Wait for installation to complete

Completion

pfSense is now installed and running in VirtualBox. The next step is configuring network adapters and integrating all VMs into a segmented lab topology.

VIRTUAL MACHINE DEPLOYMENT

Install Windows 10 on VirtualBox

Step-by-step deployment guide of a Windows 10 virtual machine on VirtualBox.

Overview

This guide explains how to deploy a Windows 10 virtual machine on VirtualBox hypervisor.

What You Will Learn

  • Download Windows 10 installation media
  • Create a Windows 10 VirtualBox VM
  • Configure storage and ISO mounting
  • Allocate virtual hardware resources
  • Complete Windows installation

Step 1 — Download Windows 10 ISO

  1. Navigate to Microsoft's Windows 10 download page.
  2. Download the Media Creation Tool.
  3. Launch MediaCreationTool_22H2.exe.
  4. Select Create Installation Media.
  5. Choose the ISO File option.
  6. Save the ISO to your VM storage location.

Step 2 — Create VirtualBox VM

  1. Open VirtualBox.
  2. Select New.
  3. Name the VM Windows10VM.
  4. Select Windows 10 (64-bit).
  5. Attach the downloaded ISO.

Step 3 — Configure Virtual Hardware

  1. Enable Unattended Installation.
  2. Configure a local username and password.
  3. Assign system resources.
  • Memory: 2048 MB
  • Processor: 1 CPU Core
  • Disk: 50 GB

Step 4 — Correct Storage Configuration

  1. Open VM Settings.
  2. Navigate to Storage.
  3. Remove the existing ISO attachment.
  4. Add an IDE Controller (PIIX4).
  5. Reattach the Windows ISO.

Step 5 — Install Windows 10

  1. Start the virtual machine.
  2. Boot from the mounted ISO.
  3. Select the appropriate region settings.
  4. Click Install Now.
  5. Select Windows 10 Home.
  6. Choose Custom Installation.
  7. Select the unallocated virtual disk.
  8. Complete the installation wizard.

Install Complete

Windows 10 is now successfully deployed and ready for network adapter configuration later.

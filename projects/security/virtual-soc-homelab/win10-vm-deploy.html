How to Install Windows 10 VM on VirtualBox

Step-by-step guide to deploying Windows 10 as a VirtualBox virtual machine and storing it in Drive D for structured lab organization.

Overview

This guide walks through installing a Windows 10 virtual machine using VirtualBox. The VM will be used as a monitored endpoint within a SOC home lab environment. If Drive D is unavailable, use the default C drive.

What You Will Learn

  • Download Windows 10 ISO using Microsoft Media Creation Tool
  • Create and configure a VirtualBox VM
  • Set correct storage and ISO mounting
  • Complete Windows 10 installation

Tip: Keep an AI Assistant Ready

VM installation issues such as boot loops, ISO errors, or storage misconfiguration are common. Having an AI assistant available can significantly reduce troubleshooting time.

Step 1 — Download Windows 10 ISO

  1. Go to: Microsoft Windows 10 Download Page
  2. Click Download Now (Media Creation Tool)
  3. Run MediaCreationTool_22H2.exe
  4. Select Create Installation Media
  5. Choose ISO File option
  6. Save ISO to D:\virtualbox-VMs

Step 2 — Create VirtualBox VM

  1. Open VirtualBox
  2. Click New
  3. Name VM: Windows10VM
  4. Select Windows 10 (64-bit)
  5. Attach downloaded ISO file

Step 3 — VM Configuration

  1. Enable Unattended Install
  2. Set Username and Password
  3. Assign Resources (Recommended):
    • RAM: 2048 MB
    • CPU: 1 Core
    • Disk: 50 GB
  4. Click Finish

Step 4 — Storage Configuration Fix

  1. Open VM Settings → Storage
  2. Remove existing ISO attachment
  3. Add IDE Controller (PIIX4)
  4. Reattach Windows ISO

Step 5 — Install Windows 10

  1. Start the VM
  2. Boot from ISO
  3. Select Region: United States
  4. Click Install Now
  5. Select Windows 10 Home
  6. Choose Custom Install
  7. Select Unallocated Disk
  8. Proceed with installation

Completion

Windows 10 VM is now successfully deployed in VirtualBox. This system will function as a monitored endpoint within the SOC lab architecture.

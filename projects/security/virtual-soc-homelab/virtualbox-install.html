Install VirtualBox on Windows 11

Step-by-step deployment guide for virtualization setup used in SOC and home lab environments.

Overview

VirtualBox is a type-2 hypervisor used to run multiple operating systems on a single host machine. This guide establishes the foundation for lab environments used in security and networking simulations.

Step 1 — Download VirtualBox

  1. Go to official VirtualBox downloads page.
  2. Select Windows Hosts installer.

Step 2 — Run Installer

  1. Open installer from Downloads folder.
  2. Run as Administrator.

Step 3 — Installation Wizard

  1. Proceed with default settings.
  2. Accept license agreement.

Step 4 — Complete Installation

  1. Install dependencies when prompted.
  2. Finish setup and reboot system.

Install Complete

VirtualBox is now ready for use to deploy the following virtual machines:

  • Kali-Linux VM
  • Windows 10 VM
  • Ubuntu server / Wazuh SIEM
  • pfSense firewall

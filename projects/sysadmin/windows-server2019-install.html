WINDOWS SERVER 2019 DEPLOYMENT

Windows Server 2019 Installation on Proxmox VE

Complete deployment and configuration guide for Windows Server 2019 within the JCAG-LAB environment. This server provides Active Directory, DNS, DHCP, SMB services, and centralized identity management.

1. Objective

Deploy Windows Server 2019 as a VM on Proxmox VE and configure it as a core infrastructure server.

  • Active Directory Domain Services (AD DS)
  • DNS Server
  • DHCP Server
  • SMB File Services
Parameter Value
VM ID 101
Hostname WIN-SVR2019
IP Address 10.10.100.11
Role Domain Controller / Core Infrastructure

2. Tools & Requirements

  • Proxmox VE installed and operational
  • Windows Server 2019 ISO
  • Proxmox Web UI access
  • RDP Client for management

3. VM Resource Planning

Component Minimum Recommended
vCPU 2 cores 4 cores
RAM 4 GB 8–16 GB
Storage 60 GB 100 GB
Network 1 NIC (VLAN 100) 1–2 NICs

4. Download Windows Server 2019 ISO

  1. Download from Microsoft Evaluation Center
  2. Upload ISO to Proxmox storage
Always verify ISO integrity before deployment.

5. Upload ISO to Proxmox

Datacenter → Storage → ISO Images → Upload

6. Create Virtual Machine (VM 101)

General Configuration

  • Name: WIN-SVR2019
  • VM ID: 101

OS Configuration

  • Select Windows Server 2019 ISO
  • Guest OS: Microsoft Windows

System Configuration

  • BIOS: OVMF (UEFI)
  • Machine: q35

CPU Configuration

  • Type: host
  • Cores: 2–4

Memory Configuration

  • RAM: 8192 MB recommended

Disk Configuration

  • Bus: VirtIO SCSI
  • Size: 80–100 GB

Network Configuration

  • Bridge: vmbr0
  • VLAN: 100
  • Model: VirtIO

7. Windows Server Installation

  1. Boot VM from ISO
  2. Select Install Now
  3. Choose Windows Server 2019 (Desktop Experience)
  4. Accept license
  5. Custom installation

8. Initial System Configuration

Rename Host

WIN-SVR2019

Static IP Configuration

Parameter Value
IP 10.10.100.11
Subnet 255.255.255.0
Gateway 10.10.100.1
DNS 10.10.100.11

PowerShell Commands

Rename-Computer -NewName "WIN-SVR2019"
Restart-Computer

9. Install Core Roles

Install-WindowsFeature AD-Domain-Services -IncludeManagementTools
Install-WindowsFeature DHCP -IncludeManagementTools

Install-ADDSForest -DomainName "jcag-lab.local" -InstallDNS

10. Post Configuration

  • Create DHCP scopes
  • Configure AD OUs
  • Create admin accounts

11. Verification

ipconfig /all
Get-ADDomain
Get-Service DNS
Get-Service DHCP

12. Summary

This server is the core identity infrastructure for JCAG-LAB providing AD, DNS, DHCP, and SMB services.

This is critical infrastructure and should be backed up regularly using Proxmox snapshots.

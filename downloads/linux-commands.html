Essential Linux Commands for Beginners

A practical command-line reference for cybersecurity, networking, and Linux system administration fundamentals.

Introduction

If you're starting your journey in Linux for cybersecurity, networking, or IT, mastering terminal commands is essential. This guide summarizes the most commonly used Linux commands in a structured and easy-to-reference format.

Navigation Commands

  • pwd → Show current directory
  • ls → List files
  • ls -l → Detailed listing
  • ls -a → Show hidden files
  • cd <directory> → Change directory
  • cd .. → Move up one level
  • cd ~ → Home directory

File & Directory Management

  • mkdir <folder> → Create folder
  • touch <file> → Create file
  • cp <src> <dest> → Copy files
  • mv <src> <dest> → Move or rename
  • rm <file> → Delete file
  • rm -r <folder> → Delete folder

Viewing & Editing Files

  • cat <file> → View content
  • less / more → Scroll file output
  • nano <file> → Simple editor
  • vim <file> → Advanced editor

Searching

  • find <path> -name "pattern" → Find files
  • grep "text" <file> → Search inside file
  • grep -r "text" <folder> → Recursive search

System Information

  • whoami → Current user
  • uname -a → System info
  • top / htop → Process monitoring
  • df -h → Disk usage
  • free -h → Memory usage

Networking

  • ping <host> → Connectivity test
  • ip a → Show interfaces
  • ss -tuln → Active connections
  • curl <url> → Fetch web data
  • wget <url> → Download files

Package Management (Ubuntu/Debian)

  • sudo apt update → Update repositories
  • sudo apt upgrade → Upgrade system
  • sudo apt install <package> → Install software
  • sudo apt remove <package> → Remove software

Permissions & Ownership

  • chmod → Change permissions
  • chown → Change ownership
  • sudo → Run as administrator

Miscellaneous

  • history → Command history
  • clear → Clear terminal
  • exit → Exit session

Practice Tip

Use a Linux VM or cloud lab environment to practice these commands safely. Repetition is key to building terminal fluency.

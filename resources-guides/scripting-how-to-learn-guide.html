Learn Scripting for Pentesting — Setup + 8-Week Study Plan

A practical structured roadmap for learning scripting for cybersecurity and penetration testing using Python, Linux, Windows, and real lab environments.

Overview

This guide provides a structured 8-week learning path focused on scripting for pentesting. It assumes a lab-based workflow using VirtualBox, Linux VMs, Windows VMs, and TryHackMe.

The goal is to develop automation skills for cybersecurity tasks such as scanning, parsing logs, SSH automation, and basic exploit scripting.

Quick Setup: Python Virtual Environment + TryHackMe

A Python virtual environment isolates dependencies so pentesting tools do not interfere with system packages.

Install Python 3

Linux / macOS

sudo apt update
sudo apt install -y python3 python3-venv python3-pip

macOS (Homebrew)

brew install python

Windows

Download Python from https://www.python.org/downloads/
Enable "Add Python to PATH"

Verify installation:

python3 --version
pip3 --version

2. Create Project Folder and Virtual Environment

Linux / macOS

mkdir -p ~/pentest-scripts
cd ~/pentest-scripts
python3 -m venv .venv
source .venv/bin/activate

Windows (PowerShell)

mkdir C:\Users\you\pentest-scripts
cd C:\Users\you\pentest-scripts
python -m venv .venv
.\.venv\Scripts\Activate.ps1

After activation, the environment will show (.venv) in the terminal prompt.

3. Install Python Packages

pip install --upgrade pip
pip install requests paramiko scapy python-nmap tqdm

4. Git and Editor Setup

git init
git add .
git commit -m "initial environment"

Use VS Code with Python extension and select the virtual environment interpreter.

6. Example Script: Log Analyzer

#!/usr/bin/env python3

from collections import Counter
import argparse, re

IP_RE = re.compile(r'(\d+\.\d+\.\d+\.\d+)')

def top_ips(path, n=10):
    cnt = Counter()
    with open(path, 'r', errors='ignore') as f:
        for line in f:
            m = IP_RE.search(line)
            if m:
                cnt[m.group(1)] += 1

    for ip, c in cnt.most_common(n):
        print(f"{ip}\t{c}")

if __name__ == "__main__":
    parser = argparse.ArgumentParser()
    parser.add_argument("file")
    parser.add_argument("-n", type=int, default=10)
    args = parser.parse_args()

    top_ips(args.file, args.n)

Run:

python log_summary.py sample.log -n 10

8-Week Study Plan

Recommended daily structure:

10m review → 30–45m lesson → 30–60m lab → 10–15m documentation

Week 1: Python Fundamentals

Variables, loops, functions, file handling, argparse, logging

Week 2: Bash & Linux Toolchain

grep, awk, sed, ssh, cron, system utilities

Weeks 3–4: Networking & Web Automation

socket programming, HTTP parsing, fuzzing, recon scripts

Week 5: SSH Automation

paramiko, remote execution, SFTP automation

Week 6: Windows PowerShell

Get-Process, event logs, remoting, CSV export

Week 7: Pivoting & Post Exploitation

proxychains, SOCKS tunneling, SSH pivoting (lab only)

Week 8: Final Project

Combine scanning, parsing, reporting, and automation into a full toolkit
